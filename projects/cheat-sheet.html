Ruby vs JS:

Defining a Hash: 

					
			# Ruby: 
			hash_name = { hash_key: hash_value , sechash_key: sechash_value}
			// JS: 
			var hash_name = { hash_key: hash_value , sechash_key: sechash_value}
~ hashes are simply called objects in JS and the keys are variables... basically the same syntax for each!

For Loops: 

					
					// JS:
					for (var i=0; i !== var_value; i++)  {
					/*actions*/
					}
					# Ruby:
					for varibale_name != variable_value do 
					 #actions
					 end
					end

While Loops: 

					
			//JS:
			while (x<10) {
 			//actions;
			x++;
			}
			#Ruby:

			end
